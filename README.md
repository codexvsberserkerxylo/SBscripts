# Roblox Spelling Bee Scripts

A collection of scripts, automated solvers, and utilities designed for various Roblox spelling bee games. This repository contains tools for rapid wordlist parsing, auto-solving, and game interaction.

## Features
* **autotype:** quickly calculates and inputs the correct spelling based on the game's prompts.
* **wordlist integration:** efficiently searches through my own custom wordlist(s) to find the current words.
* **multigame support:** built with adaptability in mind to work across different roblox spelling experiences.
